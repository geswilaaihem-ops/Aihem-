# Simple Python Port Scanner

A beginner cybersecurity project built with Python.

This tool scans a target IP address or website and checks which ports are open.

## Features

- Scans a target IP address or domain
- Detects open ports
- Uses Python socket library
- Simple command-line interface

## Example Result

Target scanned: 192.168.56.1

Open ports found:

- Port 135
- Port 139
- Port 445

## Technologies Used

- Python
- Socket
- VS Code
- Windows Terminal
